# Security Policy

## Supported Versions

The following versions are currently supported with security updates:

| Version | Supported |
|--------|-----------|
| 1.1.0   | Yes |
| older versions | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Instead, report it using one of the following:

- GitHub Security Advisories (preferred):
  Go to the repository → Security → Report a vulnerability
- Email: security@mountainviewprovisions.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigation

## Response Timeline

We aim to:
- Acknowledge reports within 48 hours
- Provide an initial assessment within 5 business days
- Release a fix as soon as reasonably possible

## Disclosure Policy
Please allow us time to investigate and address the issue before any public disclosure.
We support responsible disclosure and may credit reporters when appropriate.
